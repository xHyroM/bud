# Global config

## List of global config options

### `licenses`

```toml
[licenses]
# The default license to use when creating a new project
default = "MIT"
copyright_holder = ""

# Custom licenses
[licenses.mit]
# The name of the license
name = "MIT"
# The path to the license file
path = "LICENSE"

[licenses.apache2]
name = "Apache 2.0"
path = "LICENSE"
```
